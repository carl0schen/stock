### 關於本站

利用AI彙整台股盤後資訊，快速掃描法人當日籌碼。

### 資料來源

上市投信買賣超：感謝PTT網友整理。  
上市外資買賣超：感謝PTT網友整理。  
三大法人買賣金額統計表：感謝PTT網友整理。

### 資料彙整

使用Open AI的ChatGPT建立綜合摘要和解讀籌碼方向。

### 資料網址

網址格式為`https://stock.may.tw/西元年月日/`，例如：`https://stock.may.tw/20250905/`。資料從2025年9月5日開始上傳。  
或是到[搜尋頁面](https://stock.may.tw/search/)輸入日期前往文章。
